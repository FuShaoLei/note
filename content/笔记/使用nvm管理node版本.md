## 使用nvm管理node版本



### 安装

1. 卸载当前nodejs
2. [安装nvm](https://github.com/coreybutler/nvm-windows/releases/tag/1.1.12)
3. 配置环境变量



### 使用

查看有多少可用版本：

```
nvm list available
```

安装某一个版本：

```
nvm install 21.0.0
```

卸载某个版本：

```
nvm uninstall 21.0.0
```



查看目前安装了多少个版本：

```\
nvm list
```

使用某个版本：

```
nvm use 21.0.0
```
